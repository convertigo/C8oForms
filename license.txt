<< Software Studio >>
One of the Software core components which is for "Convertigo project" developments purposes and delivered as 
one installation file for Windows Desktops.
The Software Studio installer is installing the Software Studio.
The Software Studio includes an Eclipse IDE, Convertigo plugins, a minimal mono user Software Server, Java 
runtime documentation and installation procedure, modules such as CONVERTIGO Sequencer and SQL, REST/SOAP 
connectors. 

Projects developed with Software Studio are saved into Project.car files to be executed on "Software Servers" in 
production or on the minimal mono user Software Server included within the Studio. Project.car files including non 
Convertigo MBaaS or Mobile Builder Editions commands requires appropriate Server Configuration Servers with 
appropriate keys.

<< MBaaS Edition >>
Edition of the Software which allows You to use the Software MBaaS Edition components and configurations 
including SQL, HTML, MAINFRAME, SAP connectors, Full Sync off line data, Url Mapper and mobile SDK beyond the 
30 days' trial period and 5 Sessions with activation keys delivered by Convertigo .
The Software MBaaS Edition components are delivered and bundled with the Software Studio. They are running 
under the minimal mono user Software Server installed with the Software Studio and are free of use for 
development purpose only. 

<< Software Server >>
One of the Software core components which is for production purposes and executes Project.car files developed 
with the Software Studio.
Server is sometime named "Convertigo Enterprise Mobility Server" or "C-EMS" in documentations, web site or 
installers.
The Software Server can be configured to exploit Convertigo MBaaS and Convertigo Mobile Builder commands 
contained within project .CAR files.
The Software Server contains all run time components to run the Projects.car files produced by Convertigo MBaaS 
and Convertigo Mobile Builder. 

The Software Server installer is installing the Server Software and an administration console to monitor and 
configure Software Server.

<< Mobile Builder Edition >>
Edition of the Software which allows You to use the Software Mobile Builder Edition components and 
configurations including mobile rapid application developments beyond 30 days' trial period and 5 Sessions with 
activation keys delivered by Convertigo.
Standard Edition doesn't include Convertigo MBaaS Edition components as SQL, HTML, MAINFRAME, SAP 
connectors,Full Sync off line data, Url Mapper or mobile SDK.
The Software Mobile Builder Edition components are delivered and bundled with the Software Studio. They are 
running under the minimal mono user Software Server installed with the Software Studio and are free of use for 
development purpose only. 

<< Software SDK >>
One of the core MBaaS Edition Software component which allows to: 
-	access MBaaS Edition Servers services directly within development environments such as Xcode, Android 
studio, visual studio or Xamarin studio
-	compile native mobile applications from the aforesaid environments which can access MBaaS Editions 
Software Servers. Such mobile native applications are including the SDK Software libraries.
The SDK Software includes Convertigo proprietary developments as well as Third Party Software.
The SDK Software installed within developments environments or inside mobile device can be freely deployed with 
no limitations.
The Software SDK has installations binaries corresponding to the targeted development framework which can be 
ios android, .net, Angular and others in the future.

only be used as described in Section 7 of this license.
